## 🔖 一、资料类型标记

现在上传资料时必须选择类型啦！（我们也给所有历史资料自动标了「其他」）

| 类型 | 图标 | 说明 |
|---|---|---|
| 📚 课本 | 教材、教科书 | 课程指定参考书 |
| ✏️ 习题 | 课后练习、习题集 | 巩固知识的练习 |
| 🎯 真题 | 历年考试原题 | 期末考/考研真题 |
| 📊 课件 | 课堂 PPT / PDF | 老师上课用的课件 |
| 📝 笔记 | 学霸笔记、整理 | 学生学习笔记/总结 |
| 📋 汇总 | 知识点梳理 | 思维导图/大纲汇总 |
| 📄 其他 | 不属于以上 | 默认兜底 |

上传弹窗中新增类型下拉框（必填），管理模式下的行内编辑和批量编辑也支持修改类型。

## ❤️ 二、收藏系统

终于可以收藏喜欢的资料了！

- 文件列表每行显示 **收藏量**（第 6 列）
- 点击文件行进入**详情页** → 右上角**收藏按钮**（❤️ 空心 → 实心）一键收藏/取消
- 抽屉菜单新增 **「我的收藏」**，独立浏览历史收藏
- 通知中的「查看文件」现在**优先进入详情页**而非课程列表

---

## 📄 三、文件详情页

点击文件行的任意位置（不再是以前的直接下载），现在会打开一个**全功能的详情页面**：

**信息区**：标题、课程名称、类型标签、上传者卡片（头像/昵称/资料数/加入时间）、文件大小/类型/上传时间/下载量、描述

**操作区**：
- 📥 下载按钮（醒目）
- ❤️ 收藏切换
- ✏️ 管理模式下的内联编辑（标题/教师/描述/类型即时修改）

**预览区**：
- PDF → 浏览器内置查看器
- 图片 → 直接展示
- 文本 → 内容预览
- 其他格式 → 显示兼容提示

---

## 🔀 四、文件筛选排序

文件列表顶部新增了**筛选过滤条**和**排序下拉**：

- **类型筛选**：点击类型标签（全部/课本/习题/真题…）秒切过滤
- **排序方式**：
  - 📌 综合（默认）
  - 🕐 最新（最近上传优先）
  - 🔥 最热（下载量优先）
  - 🏷️ 类型（按资料类型分组）
  - 📦 大小（按文件大小排序）

筛选排序与分页完美集成——筛选后只显示匹配结果。
