# 搁置清单：文件管理模块（后续开发）

> 生成日期：2026-07-09
> 来源：审计报告 [design-audit-user-file-modules.md](design-audit-user-file-modules.md)
> 状态：待定，当前阶段集中构建用户模块，这些不涉及

---

## 🔴 高优先级（后续优先考虑）

| 功能 | 说明 |
|------|------|
| **PDF 在线预览** | 使用 PDF.js 或 `<iframe>` 实现浏览器内查看，无需下载到本地 |
| **文件在线预览（多格式）** | 图片、文本文档等也支持内嵌查看 |
| **拖拽上传** | 当前是传统 file input，可升级为拖拽区域 |

## 🟡 中优先级

| 功能 | 说明 |
|------|------|
| **批量操作** | 多选文件后批量下载、批量删除 |
| **文件分类标签增强** | 当前只有单维度 MaterialType，可引入多标签系统 |
| **资料评论区** | 现有点赞/点踩是轻量反馈，评论区支持自由讨论 |

## 🟢 低优先级

| 功能 | 说明 |
|------|------|
| 视频/音频播放 | 平台目前以资料文档为主 |
| 文件版本管理 | 重新上传同名文件创建新记录，不追踪版本历史 |
| 断点续传 | 50MB 限定场景下意义不大 |
| 文件二维码分享 | 社交分享加分项 |
| 真实上传进度条 | 当前是模拟进度，后续可接 xhr.upload.onprogress |
